
# Cardiac Risk ML 🫀

A machine learning web app that predicts heart disease risk using the UCI Cleveland Heart Disease Dataset 
— real clinical data from the Cleveland Clinic Foundation.

## Features
- Risk prediction using Random Forest
- SHAP explainability chart showing which factors drive your risk
- Personalised health recommendations

## Tech Stack
Python, Scikit-learn, SHAP, Streamlit, Pandas, Matplotlib

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
